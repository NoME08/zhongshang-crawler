# 中商情报网爬虫

爬取[中商情报网](https://www.askci.com)行业资讯，支持文本 + 图片采集，输出格式化的 Word/PDF 文档。

## 功能

- **多分类抓取**：覆盖产业链、市场规模、竞争格局、投融资、政策法规、产业研报、聚焦风口、产业图谱 8 个子分类
- **两种抓取模式**：按最新抓取 / 按关键词搜索
- **图文采集**：正文文本 + 图片下载
- **文档输出**：自动生成格式化 docx 文档，支持导出 PDF
- **Web 界面**：基于 Flask 的管理面板

## 环境

- Python 3.9+
- 依赖：`pip install -r requirements.txt`

## 使用

```bash
# 启动主爬虫
python main.py

# 启动 Web 管理面板
python app.py
```

配置修改见 `config.py`，可选择抓取模式、目标分类、数量等。
